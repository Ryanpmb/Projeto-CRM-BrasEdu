Projeto interdisciplinar da FATEC Rio Preto

# CRM BrasEdu
Um CRM simples, mas muito eficaz, o BrasEdu busca facilitar a conversão e o 
controle dos leads gerados.

# Tecnologias Utilizadas
Para o desenvolvimento do projeto, foram usadas as seguintes tecnologias:
  
- Java (Java 21, Spring Boot, JPA, SQL Server Driver (JDBC), Spring Security, JWT (JSON Web Token), Maven, Lombok, Bean Validation) 
- Microsoft SQL Server (Versão 2019)
- Docker

Repositório do front-end: https://github.com/Ryanpmb/braseedu-crm

## ⬇️ Como Executar
Para rodar o projeto localmente, você precisará ter o **Docker** e o **Docker Compose** instalados.

1.  **Clone o Repositório:**
    ```bash
    git clone [https://github.com/SeuUsuario/Projeto-CRM-BrasEdu.git](https://github.com/SeuUsuario/Projeto-CRM-BrasEdu.git)
    cd Projeto-CRM-BrasEdu
    ```
2.  **Inicie os Serviços:** (Back-end e SQL Server)
    ```bash
    docker-compose up --build
    ```
3.  A API estará disponível em: `http://localhost:8080`
4.  Consulte o README do front-end para iniciar a interface de usuário.
